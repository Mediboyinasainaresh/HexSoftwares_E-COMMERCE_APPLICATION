# 🛒 TradeNest - E-Commerce Android App

A modern **Flipkart-style mobile e-commerce application** built using **Kotlin, Firebase, and Razorpay integration**.

 🚀 Features

* 🔐 User Authentication (Login/Register)
* 🛍️ Product Listing with Categories
* 🔍 Smart Search & Filtering
* 🛒 Cart Management System
* 📦 Order Placement & Tracking
* 💳 Razorpay Payment Integration (Test Mode)
* 📍 Address Management
* ⭐ Product Reviews System
* ❤️ Wishlist Feature
* 🔄 Real-time Firebase Sync

🧱 Tech Stack

* **Language:** Kotlin
* **UI:** XML (Material Design)
* **Backend:** Firebase Firestore
* **Authentication:** Firebase Auth
* **Image Loading:** Glide
* **Payments:** Razorpay SDK
* 
 Screens

* Login / Register
* Home (Product Listing)
* Product Details
* Cart
* Checkout
* Order Success
* Profile

 ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/Mediboyinasainaresh/HexSoftwares_E-COMMERCE_APPLICATION.git
```

2. Open in Android Studio

3. Add your Firebase `google-services.json`

4. Run the app 🚀

 Firebase Setup

* Create Firebase Project
* Enable Firestore Database
* Enable Authentication (Email/Password)
* Add `google-services.json` in `/app`
